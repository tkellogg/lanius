# User Knowledge

This knowledge base is for durable context about the human and this elanus root.

Good pages answer questions like:

- Who is using this root and what are they trying to do with elanus?
- Why are providers, packages, agents, or policies configured this way?
- Which preferences should helpers remember across sessions?
- Which setup decisions were deliberate tradeoffs rather than accidents?

Do not duplicate raw configuration here. The actual config is already on disk and
readable through the elanus CLI. Use this KB for the reasons behind the config.
