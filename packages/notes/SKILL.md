# notes

Write markdown files under `notes/` in the harness root (create the directory
if it doesn't exist).

- One file per topic, kebab-case names: `notes/grocery-list.md`.
- Append rather than overwrite; add a `## YYYY-MM-DD` heading per addition.
- Tell the human the file path when you're done.
